# 웹 버전 빠른 실행 가이드 🌐

## 🎯 30초 안에 시작하기!

### Windows 사용자
```bash
1. 파일 다운로드
   git clone https://github.com/rpaakdi1-spec/naver-band-auto-poster.git

2. 폴더 이동
   cd naver-band-auto-poster

3. 실행 (더블클릭만 하면 됨!)
   run_web.bat
```

### Mac/Linux 사용자
```bash
1. 파일 다운로드
   git clone https://github.com/rpaakdi1-spec/naver-band-auto-poster.git

2. 폴더 이동
   cd naver-band-auto-poster

3. 실행
   python3 run_web.py
```

## ✨ 실행 후 자동으로 이런 일이 일어납니다

1. **자동 설치**: Streamlit이 자동으로 설치됩니다 (없는 경우)
2. **웹 서버 시작**: 로컬 웹 서버가 시작됩니다
3. **브라우저 열림**: 기본 브라우저가 자동으로 열립니다
4. **주소**: http://localhost:8501

## 🛠️ 필요한 것

- **Python**: 3.8 이상
- **Chrome**: 최신 버전 설치
- **인터넷**: 연결 필요

## 💡 Python 설치 확인

```bash
# Python 버전 확인
python --version
# 또는
python3 --version

# 결과 예시:
# Python 3.10.0  ✅ 3.8 이상이면 OK!
```

Python이 없다면?
- **Windows**: https://python.org 에서 다운로드
- **Mac**: `brew install python3`
- **Linux**: `sudo apt install python3` (Ubuntu/Debian)

## 🔧 문제 해결

### ❌ "python 명령을 찾을 수 없습니다"
```bash
# Windows: 환경 변수 PATH에 Python 추가
# Mac/Linux: python3 명령어 사용
python3 run_web.py
```

### ❌ "Streamlit을 찾을 수 없습니다"
```bash
# 수동 설치
pip install streamlit
# 또는
pip3 install streamlit

# 그 다음 실행
python run_web.py
```

### ❌ "브라우저가 자동으로 열리지 않습니다"
```
터미널에 표시된 주소를 수동으로 브라우저에 입력:
http://localhost:8501
```

### ❌ "포트가 이미 사용 중입니다"
```bash
# 다른 포트 사용
streamlit run streamlit_app.py --server.port 8502

# 브라우저 주소: http://localhost:8502
```

## 🎓 추가 팁

### 🌐 네트워크에서 접속하기
```bash
# 같은 네트워크의 다른 기기에서 접속 가능
streamlit run streamlit_app.py --server.address 0.0.0.0

# 출력 예시:
# Local URL: http://localhost:8501
# Network URL: http://192.168.1.100:8501  ← 이 주소로 접속!
```

### 📱 모바일에서 접속하기
```
1. PC와 휴대폰이 같은 Wi-Fi에 연결
2. Network URL 복사 (예: http://192.168.1.100:8501)
3. 휴대폰 브라우저에 붙여넣기
4. 웹 앱 사용 가능!
```
