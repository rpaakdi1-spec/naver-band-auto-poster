# 밴드 로그인 완료 확인 방법 📝

## 🎯 .exe 파일 실행 시 로그인 확인

### 방법 1: GUI 다이얼로그 (권장) ⭐

**.exe 파일로 실행 시 자동으로 사용됩니다!**

#### 화면 흐름

```
1. [시작] 또는 [수동 실행] 버튼 클릭
   ↓
2. Chrome 브라우저 자동 실행
   ↓
3. 🔐 로그인 확인 다이얼로그 창 표시
   ┌────────────────────────────────────┐
   │  밴드 로그인                        │
   ├────────────────────────────────────┤
   │  🌐 Chrome 브라우저가 열렸습니다.  │
   │                                     │
   │  📝 다음 단계를 진행하세요:        │
   │                                     │
   │  1. Chrome 브라우저에서 밴드 로그인│
   │  2. 로그인 완료 후 이 창에서       │
   │  3. [확인] 버튼 클릭               │
   │                                     │
   │  ⚠️ 로그인을 완료했나요?           │
   │                                     │
   │     [확인]        [취소]           │
   └────────────────────────────────────┘
   ↓
4. Chrome 브라우저로 이동하여 밴드 로그인
   ↓
5. 로그인 완료 후 다이얼로그 창에서 [확인] 클릭
   ↓
6. 자동 포스팅 시작!
```

#### 상세 단계

**1단계: 프로그램에서 시작 버튼 클릭**
```
[▶ 시작] 버튼 또는 [🚀 수동 실행] 버튼 클릭
```

**2단계: Chrome 자동 실행**
```
✅ Chrome 브라우저가 자동으로 열립니다
✅ https://band.us 페이지로 이동
```

**3단계: 로그인 다이얼로그 표시**
```
✅ 프로그램 창 위에 다이얼로그 팝업
✅ "밴드 로그인" 제목
✅ 로그인 안내 메시지
```

**4단계: Chrome에서 로그인**
```
1. Chrome 브라우저 창으로 이동
2. 밴드 로그인 버튼 클릭
3. 네이버 계정으로 로그인
   - 아이디 입력
   - 비밀번호 입력
   - (필요시) 2단계 인증
4. 밴드 메인 페이지 확인
```

**5단계: 다이얼로그에서 확인 클릭**
```
1. 프로그램 다이얼로그 창으로 돌아오기
2. [확인] 버튼 클릭
3. 로그인 확인 완료!
```

**6단계: 자동 포스팅 시작**
```
✅ 로그인 확인 완료 메시지
✅ 자동 포스팅 진행
✅ 로그에서 진행 상황 확인
```

### 방법 2: 콘솔 입력 (소스 코드 실행 시)

**`python run.py`로 실행 시 사용됩니다**

#### 화면 흐름

```
1. python run.py 실행
   ↓
2. Chrome 브라우저 자동 실행
   ↓
3. 콘솔에 메시지 표시:
   
   ============================================================
   🌐 Chrome 브라우저가 실행되었습니다
   📝 수동 로그인을 진행해주세요:
      1. 열린 Chrome 브라우저에서 밴드에 로그인
      2. 로그인 완료 후 프로그램으로 돌아와서
      3. Enter 키를 눌러주세요
   ============================================================
   
   ✅ 로그인 완료 후 Enter를 눌러주세요...
   ↓
4. Chrome 브라우저로 이동하여 밴드 로그인
   ↓
5. 콘솔 창으로 돌아와서 Enter 키 입력
   ↓
6. 자동 포스팅 시작!
```

## 🔧 문제 해결

### Q1: 다이얼로그 창이 표시되지 않아요

**원인**: Tkinter가 제대로 작동하지 않음

**해결**:
```
1. 프로그램 재시작
2. Windows 업데이트 확인
3. 콘솔 창에서 Enter 키로 대체
```

### Q2: [확인] 버튼을 클릭했는데 진행이 안돼요

**원인**: Chrome에서 로그인이 완료되지 않음

**해결**:
```
1. Chrome 브라우저 확인
2. 밴드 로그인 완료 확인
3. band.us 페이지에 있는지 확인
4. 다시 [수동 실행] 버튼 클릭
```

### Q3: 다이얼로그가 뒤에 숨어있어요

**원인**: 다른 창이 최상위에 있음

**해결**:
```
1. Alt + Tab으로 창 전환
2. "밴드 로그인" 창 찾기
3. 작업 표시줄에서 찾기
```

### Q4: Chrome이 자동으로 열리지 않아요

**원인**: Chrome 설치 안 됨 또는 ChromeDriver 오류

**해결**:
```
1. Chrome 브라우저 설치 확인
   https://www.google.com/chrome
   
2. 프로그램 재시작

3. 로그 확인:
   logs/ 폴더의 최신 로그 파일 확인
```

### Q5: 로그인 후에도 포스팅이 안돼요

**원인**: 로그인 세션 만료 또는 채팅방 URL 오류

**해결**:
```
1. Chrome에서 밴드 로그인 재확인
2. 채팅방 URL 정확성 확인
3. 프로그램 재시작
4. [수동 실행] 버튼으로 테스트
```
