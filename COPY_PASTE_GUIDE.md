## 🎯 **상세 설명**

### 1. 복사-붙여넣기 동작

#### **GUI 버전 (Tkinter)**

```
1. 복사 (Ctrl+C 또는 Cmd+C)
   메모장, 워드, 웹페이지 등에서 여러 줄 텍스트 복사

2. 붙여넣기 (Ctrl+V 또는 Cmd+V)
   포스트 입력창에 붙여넣기
   
3. 자동 인식 ✅
   ScrolledText 위젯이 줄바꿈(\n)을 자동으로 유지
   예: "줄1\n줄2\n줄3"

4. 포스트 추가
   [✚ 추가] 버튼 클릭
   
5. 저장
   줄바꿈이 그대로 저장됨 ✅
```

#### **웹 버전 (Streamlit)**

```
1. 복사 (Ctrl+C 또는 Cmd+C)
   메모장, 워드, 웹페이지 등에서 여러 줄 텍스트 복사

2. 붙여넣기 (Ctrl+V 또는 Cmd+V)
   포스트 입력창(text_area)에 붙여넣기
   
3. 자동 인식 ✅
   text_area가 줄바꿈(\n)을 자동으로 유지
   예: "줄1\n줄2\n줄3"

4. 포스트 추가
   [✚ 추가] 버튼 클릭
   
5. 저장
   줄바꿈이 그대로 저장됨 ✅
```

---

### 2. 전송 동작

#### **수정 전 (문제)**
```
복사-붙여넣기: "줄1\n줄2\n줄3"
                 ↓
입력: send_keys("줄1\n줄2\n줄3")
                 ↓
\n이 Enter로 해석
                 ↓
❌ 3번 전송 (줄1, 줄2, 줄3 각각 전송)
```

#### **수정 후 (해결)** ✅
```
복사-붙여넣기: "줄1\n줄2\n줄3"
                 ↓
split('\n'): ["줄1", "줄2", "줄3"]
                 ↓
전송:
  ① send_keys("줄1")
  ② send_keys(Shift+Enter)  # 줄바꿈만
  ③ send_keys("줄2")
  ④ send_keys(Shift+Enter)  # 줄바꿈만
  ⑤ send_keys("줄3")
  ⑥ send_keys(Enter)        # 전송
                 ↓
✅ 1번만 전송 (3줄이 포함된 1개 메시지)
```

## 🔍 **기술적 동작**

### GUI (Tkinter)

```python
# 1. 복사-붙여넣기 후
# ScrolledText 위젯의 내용: "줄1\n줄2\n줄3"

# 2. 포스트 추가
def add_post(self):
    content = self.post_text.get("1.0", tk.END).strip()
    # content = "줄1\n줄2\n줄3"  ✅ \n이 포함됨
    
    self.poster.config['posts'].append({
        'content': content  # ✅ 줄바꿈 유지
    })

# 3. 전송 시
def post_to_chat(self, chat_url, content):
    lines = content.split('\n')
    # lines = ["줄1", "줄2", "줄3"]
    
    for i, line in enumerate(lines):
        input_element.send_keys(line)
        if i < len(lines) - 1:
            input_element.send_keys(Keys.SHIFT, Keys.RETURN)  # 줄바꿈
    
    input_element.send_keys(Keys.RETURN)  # 전송 (1번만)
```

### Streamlit (웹)

```python
# 1. 복사-붙여넣기 후
# text_area의 값: "줄1\n줄2\n줄3"

post_content = st.text_area("포스트 내용")
# post_content = "줄1\n줄2\n줄3"  ✅ \n이 포함됨

if st.form_submit_button("✚ 추가"):
    add_post(post_content)
    # content가 그대로 저장됨  ✅ 줄바꿈 유지

# 3. 전송 시
# (동일한 로직 사용)
```
