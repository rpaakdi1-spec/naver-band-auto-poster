# 🚨 ChromeDriver 오류 즉시 해결하기

## 현재 상황
프로그램 실행 시 다음 오류가 발생합니다:
```
[WinError 193] %1은(는) 올바른 Win32 응용 프로그램이 아닙니다
```

## ✅ 즉시 해결 방법 (3가지 옵션)

---

### 🥇 방법 1: 긴급수정_실행.bat (가장 빠름)

프로젝트 폴더에서 찾기:
```
📁 naver-band-auto-poster/
   └─ 긴급수정_실행.bat  ← 이 파일 더블클릭!
```

**실행하면 자동으로**:
1. ✅ 캐시 삭제 (`C:\Users\WITHUS\.wdm`)
2. ✅ 프로그램 자동 재시작
3. ✅ ChromeDriver 자동 재다운로드

---

### 🥈 방법 2: 수동 캐시 삭제

#### Windows 탐색기 사용:

1. **Windows + R** 키 누르기
2. 다음 입력 후 엔터:
   ```
   %USERPROFILE%\.wdm
   ```
3. 열린 폴더 **전체 삭제** (Shift + Delete)
4. 프로그램 다시 실행: `실행.bat` 더블클릭

#### 명령 프롬프트 사용:

```cmd
# CMD를 관리자 권한으로 실행
rd /s /q "C:\Users\WITHUS\.wdm"

# 프로그램 재실행
cd 프로젝트경로
python run.py
```

---

### 🥉 방법 3: Python 스크립트 실행

```cmd
# 프로젝트 폴더에서
python clear_cache.py

# 또는
clear_cache.bat
```

## 🔍 문제가 계속되는 경우

### 1. Chrome 브라우저 업데이트
- Chrome 주소창에 입력: `chrome://settings/help`
- 최신 버전으로 업데이트

### 2. Python 패키지 업데이트
```cmd
pip install --upgrade selenium webdriver-manager
```

### 3. 권한 문제
- 프로그램을 **관리자 권한**으로 실행
- Windows 우클릭 → "관리자 권한으로 실행"

### 4. 방화벽/백신 확인
- 일시적으로 비활성화 후 테스트
- ChromeDriver 다운로드가 차단되었을 수 있음
