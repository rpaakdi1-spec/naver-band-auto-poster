네이버밴드 자동 포스팅 프로그램 v4.2.0
========================================

📥 설치 방법
------------

1. 네이버밴드자동포스팅.exe 파일 다운로드
2. 원하는 폴더에 저장
3. 더블클릭하여 실행

✅ 파이썬 설치 불필요!
✅ 추가 프로그램 설치 불필요!


🚀 빠른 시작
-----------

1단계: 프로그램 실행
   - 네이버밴드자동포스팅.exe 더블클릭

2단계: 채팅방 추가
   - 별명 입력 (예: "메인 채팅방")
   - 채팅방 URL 입력
   - [추가] 버튼 클릭
   - 여러 개 추가 가능

3단계: 포스트 추가
   - 포스트 내용 입력란에 메시지 작성
   - [추가] 버튼 클릭
   - 여러 개 추가 가능

4단계: 스케줄 설정
   - 포스팅 간격: 30분 (권장)
   - 랜덤 딜레이: 5분
   - 시작 일시: 원하는 시작 시간
   - 종료 일시: 원하는 종료 시간

5단계: 설정 저장
   - [💾 설정 저장] 버튼 클릭

6단계: 실행
   - [▶ 시작] 버튼 클릭
   - Chrome 브라우저 자동 실행
   - 🔐 로그인 다이얼로그 창 표시
   - Chrome 브라우저에서 밴드 로그인
   - 로그인 완료 후 다이얼로그에서 [확인] 클릭
   - 자동 포스팅 시작!


⚙️ 주요 기능
------------

✅ 다중 채팅방 포스팅
   - 여러 채팅방에 동시 포스팅
   - 체크박스로 선택적 포스팅

✅ 채팅방 별명 관리
   - 각 채팅방에 별명 지정
   - 한눈에 구분 가능

✅ 스케줄 포스팅
   - 설정한 간격으로 자동 포스팅
   - 날짜+시간 지정 가능
   - 24시간 자동 설정

✅ 실시간 카운트다운
   - 다음 포스팅까지 남은 시간 표시
   - 현재 상태 실시간 확인

✅ 로그 기록
   - 모든 활동 로그 기록
   - 문제 발생 시 확인 가능


📋 채팅방 URL 찾는 방법
----------------------

1. 밴드 웹사이트 접속
   https://band.us

2. 밴드 선택

3. 채팅 탭 클릭

4. 채팅방 클릭

5. 주소창 URL 복사
   예시: https://band.us/band/96676876/chat/CtVOZO


⚠️ 주의사항
-----------

1. Chrome 브라우저 필수
   - Google Chrome이 설치되어 있어야 합니다
   - 최신 버전 권장

2. 적절한 포스팅 간격
   - 최소 30분 권장
   - 과도한 자동화는 계정 제재 가능

3. 수동 로그인 필요
   - 보안을 위해 직접 로그인
   - 2단계 인증 지원

4. 백신 프로그램
   - 일부 백신에서 오탐지 가능
   - 실행 허용 필요


🐛 문제 해결
-----------

Q: 프로그램이 실행되지 않아요
A: Windows Defender에서 실행 허용
   - "추가 정보" 클릭
   - "실행" 버튼 클릭

Q: Chrome이 실행되지 않아요
A: Chrome 브라우저 설치 확인
   - https://www.google.com/chrome

Q: 채팅방에 포스팅이 안돼요
A: 로그인 상태 확인
   - Chrome 브라우저에서 밴드 로그인 필요
   - 로그인 후 프로그램 재시작

Q: 카운트다운이 표시되지 않아요
A: [시작] 버튼 클릭 후 대기
   - 첫 포스팅 후 카운트다운 시작

Q: 설정이 저장되지 않아요
A: [💾 설정 저장] 버튼 클릭 확인
   - config/config.json 파일 자동 생성
