## 웹 버전 🌐

### ⭐ 가장 추천! 모든 OS 지원

**장점:**
- ✅ Windows, Mac, Linux 모두 지원
- ✅ 모던한 웹 UI
- ✅ 실시간 자동 업데이트 (1초마다)
- ✅ 네트워크 접속 가능 (모바일 포함)
- ✅ 빌드 불필요
- ✅ 반응형 디자인

### 📦 필수 요구사항

```bash
Python 3.8+
pip install -r requirements.txt
```

### 🚀 실행 방법

#### Windows:
```bash
run_web.bat
```

#### Mac/Linux:
```bash
python run_web.py
```

#### 수동 실행:
```bash
pip install streamlit
streamlit run streamlit_app.py
```

### 🌐 접속

브라우저가 자동으로 열립니다:

**로컬 접속:**
```
http://localhost:8501
```

**네트워크 접속:** (다른 기기에서)
```
http://192.168.x.x:8501
```
(실행 시 터미널에 표시됨)

## .exe 파일 💻

### 📦 독립 실행 파일 (Windows 전용)

**장점:**
- ✅ Python 설치 불필요
- ✅ 단일 파일로 실행
- ✅ 더블클릭만으로 실행
- ✅ 의존성 포함
- ✅ 배포 편리

**단점:**
- ⚠️ Windows만 지원
- ⚠️ 파일 크기 큼 (50-70MB)
- ⚠️ 빌드 필요

### 📦 필수 요구사항

```
Windows 10/11 (64-bit)
Chrome 브라우저
```

Python 설치 **불필요**!

### 🔨 빌드 방법 (개발자/배포자)

#### 방법 1: 수정된 빌드 스크립트 (추천)

```bash
build_exe_fixed.bat
```

**특징:**
- 영문 이름으로 먼저 빌드
- 한글 이름으로 복사
- 더 안정적

#### 방법 2: 디버그 빌드 (오류 확인용)

```bash
build_exe_debug.bat
```

**특징:**
- 콘솔 창 표시
- 오류 메시지 확인 가능

#### 방법 3: 기본 빌드

```bash
build_exe.bat
```

### 📥 빌드 결과

```
dist/
├── BandAutoPoster.exe           (영문 이름)
└── 네이버밴드자동포스팅.exe      (한글 이름)
```

### 🚀 실행 방법

#### 사용자:

1. **다운로드**
   ```
   GitHub Releases에서 최신 .exe 다운로드
   https://github.com/rpaakdi1-spec/naver-band-auto-poster/releases/latest
   ```

2. **실행**
   ```
   네이버밴드자동포스팅.exe 더블클릭
   ```

3. **Windows Defender 경고 처리**
   ```
   "추가 정보" 클릭 → "실행" 클릭
   ```

## 💡 어떤 방법을 선택할까?

### 🌐 웹 버전을 선택하세요:

- ✅ Mac 또는 Linux 사용자
- ✅ 모바일에서도 접근하고 싶은 경우
- ✅ 모던한 UI를 선호하는 경우
- ✅ 네트워크로 원격 관리가 필요한 경우
- ✅ 실시간 자동 업데이트를 원하는 경우
- ✅ Python이 설치되어 있는 경우

### 🖥️ 데스크톱 GUI를 선택하세요:

- ✅ 전통적인 데스크톱 UI를 선호하는 경우
- ✅ Python이 이미 설치되어 있는 경우
- ✅ 독립 창으로 실행하고 싶은 경우

### 💻 .exe 파일을 선택하세요:

- ✅ Windows 사용자
- ✅ Python을 설치하고 싶지 않은 경우
- ✅ 단일 파일로 배포/사용하고 싶은 경우
- ✅ 다른 PC에 쉽게 복사하고 싶은 경우
